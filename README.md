アルクKiddyCAT市川菅野 ウェブサイト
プロジェクト概要
千葉県市川市菅野にある子ども英語教室「アルクKiddyCAT市川菅野」の公式ウェブサイト（ランディングページ）用ソースコードです。年中さんから中学3年生までを対象とし、一人ひとりに寄り添った少人数制レッスンを提供している教室の魅力を伝えるための構成になっています。

構成ファイル
index.html: メインとなるウェブページのHTMLファイル

styles.css: デザインやレイアウトを定義するスタイルシート（HTML内で読み込み）

script.js: モバイルメニューの開閉やフォームの制御などを行うJavaScriptファイル（HTML内で読み込み）

掲載コンテンツ（セクション構成）
ヒーローセクション: キャッチコピー、2026年春のキャンペーンバナー、無料体験へのボタン

特徴 (Features): 「楽しいアクティビティ」「経験豊富な講師」「季節ごとのイベント」という選ばれる3つの理由

開講クラス (Classes): 幼児コースから中学生コースまでの対象年齢とレッスン日時

料金プラン (Pricing): 4～5歳コース（月額7,150円）、小学生コース、中学生コースの料金体系と詳細説明

講師紹介 (Teacher): J-SHINE資格やTOEIC 945点を持つ講師のプロフィールと、生徒のスピーチコンテスト実績動画

無料体験 (Free Trial): 保護者様とお子様の情報、希望コース等を入力できるお申し込みフォーム

メタデータ・SEO対策
検索エンジン向けにタイトル、ディスクリプション、キーワードを設定済み

SNSシェア用にOGP（Open Graph Protocol）タグおよびTwitterカードを設定済み

Schema.orgを用いた構造化データ（EducationalOrganization）を記述し、教室の基本情報や位置情報（緯度・経度）を定義済み
